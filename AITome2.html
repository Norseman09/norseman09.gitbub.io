  • The All Mighty Xanathar

    A chatbot linked to the DnD5e API

    This program showcases the application of artificial intelligence through a chatbot designed to engage users and provide information on Dungeons & Dragons game elements. By utilizing the requests library for API calls and processing the obtained data, the AI-driven chatbot demonstrates its ability to analyze and integrate information seamlessly. It makes HTTP requests to the D&D 5e API, retrieves data, and processes it to provide information about classes, races, and spells. The bot parses the API response and displays relevant information about classes, races, and spells, showcasing an understanding of natural language and data processing.

    Figure 1.1 - Showing the programing running.

    Figure 1.2 - The shown code snippet is a Python script designed to interact with the Dungeons & Dragons 5th Edition (D&D 5e) API. It defines functions to retrieve information about D&D classes, races, and spells by making GET requests to the API. The get_data function fetches data from the API, and two specific functions, get_class_info and get_race_info, utilize this to obtain details about D&D classes and races, respectively. Notably, the script includes a check in the get_race_info function to handle the format of ability bonuses, converting them from a list to a dictionary if necessary. Additionally, there's a get_spell_info function for fetching information about D&D spells. The script concludes with a chat function that initiates interaction with the user, providing instructions and a chatbot interface for querying information about races, classes, and spells in the D&D 5e world.

    Figure 1.3 - The provided Python script implements a chatbot named "The All Mighty Xanathar," engaging users in inquiries about Dungeons & Dragons (D&D) classes, races, and spells. Utilizing a perpetual loop, the chatbot prompts users to input queries such as "class," "race," or "spell," processing requests through specific functions linked to the D&D 5e API. The script includes a graceful exit option with the command "exit." Humorous error messages emphasize that if The All Mighty Xanathar is unaware of something, then it simply does not exist. This code showcases a straightforward yet interactive text-based interface, offering information retrieval from the D&D 5e API with an engaging tone.

  • Ioun: The Goddess of Knowledge Chatbot

    Unlike the command-line interface of the Xanathar bot, this chatbot has a graphical user interface (GUI) created using the Tkinter library. It engages users through a more user-friendly visual interface.The chatbot incorporates a text area that displays both user inputs and model responses, creating a conversational history. This feature enhances user interaction and makes the conversation more understandable. The chatbot utilizes a language model (Llama) to generate responses. It tokenizes user inputs, sends them to the model, and displays the model's responses in the GUI.
